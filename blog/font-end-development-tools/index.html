前端开发相关工具和资源

发布于2020年5月23日

Chrome 扩展

  • Wappalyzer Wappalyzer 可以识别网站使用的技术、内容管理系统，JavaScript 框架，电商平台，内容分发系统以及营销和分析工具等。
  • Octotree 使用侧边栏菜单导航 github 仓库文件。
  • GitHub Repository Size 显示 github 仓库文件大小。
  • Web Developer 该扩展增加了一个具有多种网页开发者工具的功能条。比如它可以禁用 JavaScript，禁用 CSS 样式
  • Site Palette 一键提取网页配色
  • Fonts Ninja 识别网站上所有文本的字体。
  • JSON Viewer 在浏览器上高亮显示 JSON。

在线应用

  • transform 多种 Web 语言转换器，比如 HTML to JSX，JSON to YAML
  • SVGOMG SVG 文件优化在线应用
  • Dynamic Dummy Image Generator 图片占位符自动生成器
  • Lorem Picsum 图片自动生成器，图片来自 Unsplash
  • Easing functions 贝塞尔曲线库
  • cubic-bezier 在线调试贝塞尔曲线
  • Get emoji 表情资源，可复制粘贴使用
  • prism-break 隐私/安全软件
  • RapidAPI 提供了很多领域的 API，有免费和付费之分
  • Modular Scale 字型大小比例尺
  • websequencediagrams 时序图
  • regular expression 101 在线调试正则表达式，支持多种语言
  • WebGradients WebGradients 免费提供了 180 种线性渐变配色方案。
  • uiGradients uiGradients 线性颜色渐变集合。
  • Can I use 浏览器兼容性查询，支持查询 HTML CSS SVG HTML 5CSS3 特性。
  • keyframes 在线创造动画，阴影和颜色
  • extendsclass - Online Tools for Developers

github 资源

  • GitHub Topic 浏览 github 中受欢迎的项目
  • chromeapp Chrome 扩展推荐
  • shields Quality metadata badges for open source projects
  • HELLOGITHUB 分享 GitHub 上有趣、入门级的开源项目

  • REST Countries

  • gitpod 建立云端开发环境
